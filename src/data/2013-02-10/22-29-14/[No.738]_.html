Mixed In Key Review: A Harmonic Mixing Tool

Friday February 1, 2013

Whether you use crossfading in your favorite software media player (like iTunes) to listen to digital music, or use DJ software to create remixes from MP3s, harmonic mixing is a must for getting perfect transitions. Rather than having to know music theory and spend hours matching songs by ear, Mixed In Key could just save you heaps of time.

It works by analyzing the audio files in your library and detecting the key (sometimes called root key), tempo (BPM), and energy level. This is essential for seeing what songs work together in your collection. In fact, top DJ's and musicians use harmonic mixing in order to avoid bad sounding key clashes.

In the latest version of Mixed In Key, there's also a energy level feature which indicates how 'danceable' a track is. This is useful for predicting how an audience or crowd will react to your music.

For a full look at this software tool, read our Mixed In Key review for the full lowdown.

Configure iTunes to Automatically Update Playlists Based on Song Ratings

Wednesday January 30, 2013

Are you struggling to keep your iTunes library organized? There are quite a few ways to make the task of sorting and organizing the contents of your iTunes library easier. However, the star rating system probably isn't the first tool you'd think of for this task.

After all, doesn't it just show you the best and worst of your iTunes library?

By using the 5-star song rating system in iTunes you can also create self-updating playlists -- called Smart Playlists. This makes it far easier for instance to keep an up-to-date playlist of your most favorite songs. As your library changes, and you rate songs, these Smart Playlists will also change. These dynamic playlists based on star ratings can also be used for other tasks too, like syncing only the best content to your Apple device.

To see how to create this type of playlist, follow our short iTunes tutorial on using song ratings in Smart Playlists.

Add New Features to Windows Media Player by Using Free Plugins

Wednesday January 23, 2013

Plugins or add-ons as they are sometimes called enable you to extend the capabilities of a software program. Windows Media Player (WMP) supports plugins which are useful bolt-on applications for adding new features and customizing the interface for your particular needs.

For example, if you use Windows Media Player 12 and miss the advanced tag editor (previously an option in WMP 11), then you can add this facility by using a plugin. Another example of a plugin that you might want to install is one that displays lyrics on-screen in real-time. If you want to learn the words to a song, then this can be a useful addition to WMP for visually learning a song rather than by ear alone.
