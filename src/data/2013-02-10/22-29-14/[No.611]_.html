C++ Tutorial on Vectors published

Sunday February 10, 2013

This is the fifth C++ programming tutorial in the series and looks at using the vector template class to follow on from arrays in tutorial four.

I've always liked the vector template class. It's easy to use and gives more reliable code if you use the .at() to indexthe vector rather than the [] syntax. Depending on your need though the List and Array template classes may be more suited. I'll be looking at those in the next tutorial.

First C# SDL.NET Tutorial Published

Friday February 8, 2013

I'd been neglecting the C# games world for a while so after looking into it, I've created a simple demo using the SDL.NET library and written a programming games in C# with SDL.NET tutorial about it.

I'm very impressed with all the features the SDL.NET library provides. Things like sprites, including animation and text, surfaces for 2D and OpenGl, support for Movie loading and playing, bezier, polygon (and textures), square, circle, line, pie drawing primitives, particle support with emitters and sprites and manipulators and a bit more.

Boost 1.5.3 Released Plus Ebook

Wednesday February 6, 2013

The open source library Boost latest version was released a couple pf days ago. Now up to 1.5.3, this adds five new libraries: Atomic, Coroutine, Lockfree, Multiprecision and Odeint and updates to some thirty others.

Boost is an excellent library that adds considerable functionality to the STL (Standard Template Library). If you want to get started with it, you could take a look at the free online Boost C++ Libraries Ebook, or buy the printed version which is newer.
